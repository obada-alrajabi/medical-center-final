لكل قسم فرعي مذكور أعلاه، أضف داخله صلاحيات تفصيلية قابلة للتفعيل/التعطيل بـ Toggle:
فتح ملف مريض: عرض، تسجيل جديد، تعديل البيانات، تعديل التاريخ، طلب حذف، طباعة
قائمة الانتظار: عرض، إضافة، تعديل الحالة، حذف
طلبات (مختبر/أشعة/تأهيلي) وتعبئة النتائج: عرض، إنشاء جلسة، تعبئة النتائج، طباعة
دليل الأسعار: عرض، إضافة، تعديل، حذف
مخزون / مستلزمات: عرض، إضافة، تعديل، حذف
قوائم الحالات: عرض فقط
طلبات الشراء: عرض، إنشاء طلب
الطباعة والتصدير: طباعة التقارير
دوام الموظفين: عرض السجل، تسجيل حضور/انصراف
السندات وحساباتها: عرض، إنشاء سند قبض، إنشاء سند صرف
الربح: عرض تقارير الربح
الديون: عرض، تسوية
الإيرادات / الدخل: عرض
المصروفات: عرض، إضافة
المطلوب تقنياً:

أضف جدول staff_subitem_permissions بقاعدة البيانات:

sqlCREATE TABLE staff_subitem_permissions (
  id SERIAL PRIMARY KEY,
  staff_id INTEGER NOT NULL REFERENCES staff_members(id) ON DELETE CASCADE,
  dept VARCHAR(50) NOT NULL,
  subitem VARCHAR(100) NOT NULL,
  permission VARCHAR(100) NOT NULL,
  granted BOOLEAN DEFAULT FALSE,
  UNIQUE(staff_id, dept, subitem, permission)
);

أضف Endpoints:

GET /api/staff/:id/subitem-permissions
POST /api/staff/:id/subitem-permissions
