 [🚨 أمر سيادي حاسم - تشغيل محرك الطباعة والتصدير الفعلي وتفعيل كافة خيارات اللوحة 100%]:
مرحباً Claude. هذا أمر برامجي صارم، ومحدد النطاق، وذو أولوية قصوى لتشغيل وتفعيل نظام "الطباعة والتصدير" وتحويل كافة الخانات والأزرار الظاهرة في الواجهات إلى أزرار فعلية وشغالة تماماً بنسبة 100% (Functional Print Engine Deployment).

⚠️ تنبيه أمني برامجي صارم للغاية: يُمنع منعاً باتاً، وتحت أي مسمى أو ظرف، التعديل أو التغيير أو المساس بشجرة الصلاحيات، دالة حذف السندات وأثرها المالي، أو نظام فرز دليل أسعار المختبر (الداخلي والخارجي) المعتمد سابقاً. التعديل محصور حصراً في ربط وتشغيل لوحة وأزرار الطباعة والتصدير عبر النظام بأكمله.

❌ تحذير حاسم: يُمنع منعاً باتاً ترك أي زر أو خانة كـ "منظر" أو واجهة ثابتة. يجب أن يتأثر ملف الـ PDF المطبوع أو المصدر بكل خيار يتم تحديده في اللوحة الحالية حرفياً.

يرجى تفعيل وتشغيل المنطق البرمجي فوراً وفق المحددات التقنية التالية بدقة 100%:

### 🎛️ 1. تفعيل الخيارات التشغيلية للوحة الطباعة والتصدير (Active Control Binding):
قم بربط الـ State الخاص بجميع حقول المدخلات في اللوحة لتعكس التغييرات فوراً عند الضغط على زر "طباعة" أو "حفظ PDF" كالتالي:
1. *📌 مع ترويسة / بدون ترويسة (Header Toggle):* عند اختيار "بدون ترويسة"، يتم إخفاء الترويسة تماماً. وعند اختيار "مع ترويسة المركز"، يتم تفعيل الترويسة.
2. *🖼️ صورة الترويسة الرسمية (Full-Page Letterhead Upload):* عند رفع الصورة، يجب برمجياً حقنها عبر الـ CSS @media print كخلفية كاملة وممتدة تغطي كامل ورقة الـ A4 المطبوعة (background-image بمقاس A4 كامل، و background-size: cover أو 100% 100%) لتظهر كافة النصوص والبيانات الطبية والمالية المحددة فوقها بدقة بالغة وليس على شكل ترويسة علوية فقط.
3. *📐 هوامش الصفحة بالمليمتر (Dynamic Margins Input):* قم بربط الخانات الأربعة (العلوية، السفلية، الأيمن، الأيسر) مباشرة بـ CSS Page Margin. بمجرد تغيير الموظف للأرقام (مثلاً: العلوي 25mm، الأيمن 15mm)، يتم تطبيقها تلقائياً داخل قاعِدة @page { margin: ... } الخاصة بأمر الطباعة الفعلي للمتصفح لضمان عدم خروج النصوص عن الهوامش المطلوبة.
4. *🔤 إعدادات متقدمة ونمط الخط (Font Customization):* يجب أن يتغير نوع الخط وحجمه في التقارير والجلسات بناءً على القيم المختارة في القائمة المنسدلة للوحة فوراً.

### 🧾 2. ربط تحديد الجلسات والحالات (Dynamic Records Selection):
- في الجدول الأيمن (جدول الزيارات والجلسات مثل جلسات مريض الجراحة أو الفحوصات)، يجب تفعيل الخانات (Checkboxes) لـ "تحديد الكل" أو تحديد جلسات معينة.
- عند الضغط على زر طباعة أو حفظ PDF، يقوم النظام بفلترة وتجميع بيانات الجلسات "المحددة فقط" وتضمين تفاصيلها الطبية والتشخيصية بالكامل داخل نموذج الطباعة، وحظر طباعة أي جلسة لم يتم تحديدها.

### 💾 3. تشغيل أزرار الإجراءات العلوية فعلياً (Print & Export Actions):
- *🖨️ زر طباعة (Print Button):* عند النقر عليه، يتم تجميع الـ HTML المخصص للتقرير مع حقن الـ CSS الديناميكي المأخوذ من مدخلات اللوحة (الهوامش، الخلفية، الخط)، ثم إطلاق أمر الطباعة الفعلي للمتصفح window.print() مباشرة وبشكل مستقر.
- *📄 زر حفظ PDF (Save PDF Button):* عند النقر عليه، يتم توليد وتصدير ملف الـ PDF بنفس التنسيقات الدقيقة والهوامش المحددة باستخدام محرك التصدير المعتمد في النظام (مثل html2pdf أو jspdf) وتحميله فوراً لجهاز المستخدم.

⚠️ تأكيد ختامي ملزم: تقيد بالتشغيل البرمجي الفعلي الكامل لكافة عناصر وأزرار لوحة الطباعة والتصدير عبر النظام بأكمله ومطابقتها للقطات الشاشة المرفقة دون زيادة أو نقصان، وأبقِ كل شيء آخر في النظام على حاله بنسبة 100%، واعرض لي الأكواد المحدثة والشغالة فوراً.
